# Generador Masivo de Gafetes — Riverline Ergonomic

## Cómo usar

1. Abre index.html
2. Sube tu archivo Excel
3. Sube las fotos
4. Da clic en 'Generar Gafetes'
5. Se descargará automáticamente un ZIP

## Columnas del Excel

nombre
puesto
unidad
fecha
id
foto

## Importante

La columna FOTO debe coincidir con el nombre del archivo.

Ejemplo:
Excel: juan
Archivo: juan.jpg
